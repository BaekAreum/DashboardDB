# DashboardDB
## 프로젝트 소개
데이터베이스를 연결하여 해당 데이터를 대시보드로 확인하는 백엔드 코드입니다.
<br>

## 개발 환경
- `java 16`
- `jdk 16.0.2`
- **IDE** : EclipseIDE(eGovFrame)
- **Framework** : egovFrame4.0
- **Database** : MariaDB
